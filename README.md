# music_streaming_service

### Features:

  1. Nested playlist
  2. Collaborative playlist
  3. Custom sorting of playlists
  4. Radio station
  5. Incognito mode
  6. Song queue system
  7. Song ratings
  8. Song collaboration
  9. User listening history
  10. Artist following system

### Stack: MySQL, Node.js, Express.js, React.js
